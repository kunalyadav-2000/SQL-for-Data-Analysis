# Introduction to SQL

## Overview
SQL (Structured Query Language) is the universal language for communicating with relational databases. As a data analyst, SQL is your primary tool for extracting, transforming, and analysing data stored in tables.

## Why It Matters
- **Every company** uses databases to store business data (sales, customers, inventory). SQL is the bridge between raw data and decision‑making.
- In Data Analysis, SQL allows you to answer business questions like “What were last month’s top‑selling products?” or “Which customers haven’t purchased in 90 days?”
- It is consistently ranked as the #1 skill for data analyst roles.

## Real‑World Relevance
- An e‑commerce analyst runs a daily SQL query to pull the previous day’s revenue.
- A marketing analyst joins customer and campaign tables to measure ROI.
- A product analyst examines user events to understand feature adoption.

## Learning Objectives
- Understand what SQL is and where it fits in the data ecosystem.
- Recognise the difference between SQL as a language and the database engine that runs it.
- Appreciate the role of SQL in business analytics.

## Key Concepts
- **SQL** – a declarative language, meaning you describe *what* you want, not *how* to get it.
- **Database Engine** – software that executes SQL queries (e.g., PostgreSQL, MySQL).
- **CRUD** – Create, Read, Update, Delete – the four fundamental data operations.

## Interview Questions
1. **Beginner:** What does SQL stand for?  
   *Answer:* Structured Query Language.
2. **Intermediate:** Is SQL a programming language?  
   *Answer:* SQL is a domain‑specific language, not a general‑purpose programming language. It is declarative and used primarily for database querying.
3. **Scenario:** A manager asks you to “pull last quarter’s sales numbers.” How would you approach it?  
   *Answer:* Identify the relevant tables (orders, order_items), filter by date using WHERE, and aggregate the amount using SUM, grouping by month or category as needed.

## Summary
SQL is the foundation of all data analysis. Every business question that involves structured data can be answered with SQL.
