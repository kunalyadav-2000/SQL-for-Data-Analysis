> A complete, project‑based SQL curriculum designed to take you from beginner to **Data Analyst job‑ready**. Every concept is backed by real‑world business cases and documented as a polished GitHub portfolio.

## 🚀 Project Overview

This repository is both a **learning roadmap** and a **portfolio**. You’ll master SQL step‑by‑step and simultaneously build 10 analytical projects that demonstrate your ability to solve business problems with data.

## 🎯 Learning Objectives

- Write efficient, complex SQL queries for data analysis
- Perform cohort, retention, funnel, and RFM analysis
- Optimise queries using execution plans and indexing
- Build professional‑grade projects that impress recruiters
- Prepare for FAANG‑level SQL interviews

## 🛠️ Skills Covered

- **SQL Dialects**: PostgreSQL, MySQL, SQL Server (adaptable)
- **Analytical Techniques**: Cohort, Retention, Funnel, RFM
- **Business Metrics**: MRR, ARR, LTV, CAC, Churn, AOV
- **Optimisation**: Indexing, Execution Plans, Partitioning
- **Tools**: GitHub, Markdown, DB Diagram, db-fiddle, Tableau (optional)
